# Impeccable 스킬 전체 목록

| 스킬 | 설명 |
|------|------|
| **`impeccable:adapt`** | 다양한 화면 크기, 기기, 플랫폼에서 일관된 경험을 보장하도록 디자인을 적응시킴 |
| **`impeccable:arrange`** | 레이아웃, 간격, 시각적 리듬 개선. 단조로운 그리드, 불일치 간격, 약한 시각적 계층구조를 수정 |
| **`impeccable:animate`** | 기능에 목적 있는 애니메이션, 마이크로 인터랙션, 모션 효과를 추가하여 사용성과 즐거움 향상 |
| **`impeccable:audit`** | 접근성, 성능, 테마, 반응형 디자인 전반에 걸친 종합 감사. 심각도별 이슈 리포트 생성 |
| **`impeccable:colorize`** | 단조로운 인터페이스에 전략적 색상 추가. 더 생동감 있고 표현력 있게 만듦 |
| **`impeccable:clarify`** | 불명확한 UX 문구, 에러 메시지, 마이크로카피, 라벨, 안내문 개선 |
| **`impeccable:bolder`** | 안전하거나 지루한 디자인을 시각적으로 더 자극적이고 임팩트 있게 증폭 |
| **`impeccable:critique`** | UX 관점에서 디자인 효과성 평가. 시각적 계층, 정보 구조, 감성, 전반적 품질에 대한 피드백 |
| **`impeccable:delight`** | 즐거움, 개성, 예상치 못한 터치를 추가하여 기능적 인터페이스를 기억에 남는 경험으로 |
| **`impeccable:distill`** | 불필요한 복잡성 제거. 디자인을 본질로 정제 — 단순하고 강력하고 깔끔하게 |
| **`impeccable:extract`** | 재사용 가능한 컴포넌트, 디자인 토큰, 패턴을 디자인 시스템으로 추출 및 통합 |
| **`impeccable:frontend-design`** | 고품질 프론트엔드 인터페이스 생성. 웹 컴포넌트, 페이지, 앱 등 제작 시 사용 |
| **`impeccable:harden`** | 에러 처리, i18n, 텍스트 오버플로, 엣지 케이스 관리로 인터페이스를 프로덕션 수준으로 강화 |
| **`impeccable:normalize`** | 디자인 시스템에 맞춰 일관성 있게 정규화 |
| **`impeccable:onboard`** | 온보딩 플로우, 빈 상태, 첫 사용자 경험 설계/개선 |
| **`impeccable:optimize`** | 로딩 속도, 렌더링, 애니메이션, 이미지, 번들 크기 등 성능 최적화 |
| **`impeccable:overdrive`** | 셰이더, 60fps 가상 테이블, 스프링 물리, 스크롤 기반 효과 등 기술적으로 야심찬 구현 |
| **`impeccable:polish`** | 출시 전 최종 품질 패스. 정렬, 간격, 일관성, 디테일 수정 |
| **`impeccable:quieter`** | 지나치게 강렬하거나 공격적인 디자인을 톤다운. 품질은 유지하면서 강도 조절 |
| **`impeccable:teach-impeccable`** | 프로젝트의 디자인 컨텍스트를 수집하여 AI 설정 파일에 저장하는 1회성 셋업 |
| **`impeccable:typeset`** | 폰트 선택, 계층, 크기, 굵기 일관성, 가독성 등 타이포그래피 개선 |

## 워크플로 예시

감사(audit) → 비평(critique) → 개선(colorize, arrange, animate 등) → 마무리(polish)
