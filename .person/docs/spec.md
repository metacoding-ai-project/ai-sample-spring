# 기능 명세서 (Functional Specification)

## 1. 회원 관리 (User Management)
- **F-1.1 회원가입**
    - 사용자는 아이디, 비밀번호를 입력하여 새로운 계정을 생성할 수 있다.
    - 중복된 아이디로 가입할 수 없다.
- **F-1.2 로그인**
    - 사용자는 가입된 정보를 바탕으로 시스템에 인증을 수행할 수 있다.
    - 성공 시 세션 또는 JWT 기반의 인증 정보를 획득한다.
- **F-1.3 회원 탈퇴**
    - 사용자는 자신의 계정을 영구적으로 삭제할 수 있다.
    - 탈퇴 시 해당 사용자가 작성한 모든 게시글과 댓글은 즉시 비노출(또는 물리적 삭제) 처리되어야 한다.

## 2. 게시글 기능 (Post Management)
- **F-2.1 게시글 목록 조회**
    - 모든 사용자는 최신순으로 정렬된 게시글 목록을 확인할 수 있다.
    - 페이징 처리가 적용되어야 한다.
- **F-2.2 게시글 상세 조회**
    - 게시글의 전체 본문과 함께 등록된 모든 댓글 목록을 조회할 수 있다.
- **F-2.3 게시글 작성**
    - 로그인한 회원만 텍스트 기반의 게시글을 작성할 수 있다. (이미지/에디터 제외)
- **F-2.4 게시글 수정**
    - 작성자 본인만 게시글의 내용을 수정할 수 있다.
- **F-2.5 게시글 삭제**
    - 작성자 본인만 게시글을 삭제할 수 있다.
- **F-2.6 게시글 검색**
    - 제목 또는 본문의 키워드를 사용하여 게시글을 필터링하여 검색할 수 있다.

## 3. 댓글 기능 (Comment Management)
- **F-3.1 댓글 작성**
    - 로그인한 회원만 특정 게시글에 댓글을 작성할 수 있다.
- **F-3.2 댓글 삭제**
    - 작성자 본인만 자신이 작성한 댓글을 삭제할 수 있다.
- **F-3.3 댓글 구조**
    - 대댓글을 지원하지 않는 1단 계층 구조를 유지한다.
